## 🔧 How to Use

1. **Update Social Links**: Replace placeholder GitHub/LinkedIn URLs in:
   - `HeroSection.tsx`
   - `Footer.tsx`

2. **Customize Testimonials**: Edit testimonials in `TestimonialsSection.tsx`

3. **Add Resume**: Link your resume PDF in the navbar download button

4. **Adjust Colors**: Modify CSS variables in `index.css` for custom theming
